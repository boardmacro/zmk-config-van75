### ‚úÖ Langkah Menimpa Isi GitHub dengan Folder Lokal `D:\bt75\`

#### 1. **Buka Command Prompt dan masuk ke folder**
```cmd
=============================================
cd /d D:\bt75
```
=============================================
#### 2. **Pastikan folder sudah jadi repo Git**
Kalau belum ada `.git`, jalankan:
```cmd
=============================================
git init
```
=============================================
#### 3. **Set remote GitHub**
Ganti URL dengan milikmu:
```cmd
=============================================
git remote add origin https://github.com/boardmacro/zmk-config-van75.git
=============================================

Kalau sudah ada remote, bisa reset:
```cmd
=============================================
git remote set-url origin https://github.com/boardmacro/zmk-config-van75.git
```
=============================================
#### 4. **Tambahkan semua file dan commit**
```cmd
=============================================
git add .
git commit -m "Menimpa isi GitHub dengan backup dari D:\\bt75"
=============================================

```
=============================================
#### 5. **Force push ke GitHub**
Jika branch utama kamu adalah `master`:
```cmd
=============================================

git push origin master --force
=============================================
```
Atau jika pakai `main`:
```cmd
=============================================
git push origin main --force
=============================================
```

---

### üõ°Ô∏è Catatan Penting
- `--force` akan **menghapus histori commit lama** di GitHub dan menggantinya dengan commit dari lokal.
- Pastikan kamu punya akses *write* ke repo GitHub tersebut.
- Setelah push, GitHub Actions akan otomatis jalan jika sudah dikonfigurasi.

---

Kalau kamu mau, kita bisa buat **tag versi** seperti `v1-backup-dbt75` agar kamu bisa balik ke versi ini kapan saja. Atau bikin branch khusus untuk backup sebelum menimpa `master`. Mau aku bantu setup itu juga?
